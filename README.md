# numpy - a Linear Algebra Library for Python (conda install numpy or pip install numpy)
- 2 types of NumPy arrays
  - vectors - 1-d arrays
  - matrices - 2-d arrays
```
import numpy as np
my_list = [1,2,3]
arr = np.array(my_list)

my_mat = [[1,2,3],[4,5,6],[7,8,9]]
mat = np.array(my_mat)
```
